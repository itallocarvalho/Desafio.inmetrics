
Cadastro de Livros:

Enviar via POSTMAN ou outro programa que realiza comunicação via HTTP, 
uma requisição POST para a URL:

[localhost:8080]/livro/cadastrar

Utilizar o json abaixo no body:

{
  "id": 1,
  "title": titulo do livro",
  "description": "descrição do livro",
  "pageCount": 100,
  "excerpt": "resumo do livro",
  "publishDate": "2019-03-22T00:02:34.716Z"
}
